Chrome改User Agent

方法一 通过在开发者工具里设置单一网站的访问 UA

这个方法只适用于单一网页，我觉得不大好用。参考链接1的作者也说：注意：这种方式修改只能在当前 tab 生效，新建 tab 时仍旧是默认 UA。

1 打开开发者工具

2 打开Network conditions

3 设置User Agent

取消勾选 Select automatically，然后就可以在下面文本框里手输User Agent了。完事后刷新下就OK。

方法二 Toggle Device Bar

1 打开开发者工具，激活Toggle Device Bar

按下按钮，使之变蓝

2 在上方新出现的一栏中选择你想要的设备，如果没有，那就新建一个

新建Device

END
