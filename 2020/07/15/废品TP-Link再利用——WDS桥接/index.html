废品TP-Link再利用——WDS桥接

發表於 2020-07-15 | 分類於 路由器 | | 閱讀次數:
字數統計: 1.1k | 閱讀時長 ≈ 3

因为装修时没有考虑到日后网络使用的问题，导致后来装宽带时才发现入户的网线居然是断的，简单粗暴的钻墙从户外新拉了根光纤入户后，还是没能用上原来装修时室内布的网线（没搞懂弱电箱里那玩意儿该怎么用），而且网线的插座设置只考虑到了有线网络的使用，位置都在房子的边边角角，没有合适的、对无线友好的网线插口，只能把无线路由器放在家里最西边弱电箱前（新拉的光纤是通过弱电箱里钻墙打洞的孔里入户的），导致家里靠东一块信号几乎没有。忍受了多年的无线网络后，我终于还是购入了一个Redmi AX5，把服役多年的TP-Link TL-WR745N换下了。

虽说现在室内的确都有信号，但我手上还有个奇葩iPhone，在房子靠东一侧还是无法使用WiFi。看到换下的TP-Link支持WDS，正好，就让你再发挥余热吧。

TP-Link TL-WR745N WDS配置过程

正文

1、修改管理 IP 地址

登录路由器的管理界面，点击 网络参数 >> LAN 口设置，修改 IP地址（如主路由器的 IP 地址是 192.168.1.1，则副路由器修改为 192.168.1.2），保存并重启路由器。

注意：重启后使用修改后的 IP 地址登录副路由器的管理界面。

2 、修改无线设置及开启 WDS 扫描信号

使用修改后的 IP 地址登录路由器管理界面，点击 无线设置 >> 无线基本设置，设置与主路由器相同的 SSID 号 及 信道，勾选 开启 WDS，点击 扫描。

3 、选择主路由器信号

在扫描页面中找到主路由器的无线 SSID，并点击 连接。

注意 ：如果扫描不到主路由器的信号，请确认前端主路由开启无线功能，且尝试减小主、副路由器之间的距离。

4 、输入密码

在 密钥 位置输入主路由器的无线密码，输入完成后点击 保存。

5 、设置副路由器的无线密码

点击 无线设置 >> 无线安全设置，选择与主路由器相同的加密方式，在 PSK 密码 中设置与主路由器相同的无线密码，点击 保存。

注意： 副路由器的 SSID、无线密码可与主路由器不同，如果需要实现漫游，则必须与主路由器相同。

6 、关闭 DHCP 服务器

点击 DHCP 服务器 >> DHCP 服务，DHCP 服务器选择为 不启用，点击 保存。

注意 ：DHCP 关闭成功后，请重启路由器，重启的方法：点击 系统工具 >> 重启路由器，点击重启路由器。

重启完成后，在路由器的管理界面，点击 运行状态，观察 无线状态 >> WDS 状态，显示 成功 则表示桥接 WDS 设置成功。

至此无线桥接（WDS）设置完成，无线终端连接 zhangsan，有线电脑连接主、副路由器的 LAN 口即可上网。终端移动过程中，可以实现主、副路由器间的自动漫游。

注意： 无线桥接（WDS）设置完成后，副路由器只需通电即可，无需其他设置。

Caution

  1. 首先要在网络参数中设置好副路由器的IP地址，要根据主路由器的IP设置（除非你一次性设置OK并不想再碰副路由器的设置面板，不然本步骤非常重要）
  2. 建议进行中间步骤时尽量不要重启路由器，不然可能前功尽弃
  3. 建议改掉Dashboard的管理账号，并为路由器设置时间

后记

设置完WDS后，iPhone网络还是时断时续。我事后查看TP-Link的Dashboard发现无限状态中WDS状态经常在初始化和成功之间跳来跳去，原因似乎是副路由器常常搜不到主路由器的信号……好吧，老玩意儿还是不好用啊，我的支持WiFi 5的笔记本直连主路由器网络都很稳定的说😂
